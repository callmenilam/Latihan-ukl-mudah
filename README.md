Deskripsi : program ini digunakan untukmenghitung biaya pengiriman paket bedasarkan
            1.Jarak tempuh ( dalam kilometerr).
            2.berat paket (dalam kilogram)
            3. Ukuran aket (panjang , lebar , tinggi dalam satuan Cm )
           Program ini akan menghitung volume,menentukan biaya perkilogram ,serta menambahkan biaya tamabahan jika volume paket terlalu besar.
Cara kerja: 1. PENGGUNA MEMASUKKAN:
                1.Jarak(Km)
                2.Berat(Kg)
                3.Panjang,lebar,dan tinggi paket
            2.  PROGRAM MENGHITUNG VOLUME = p x l x t
            3.MENENTUKAN BIAYA PER KG:
                1. Jika jarak < 10 km -->Rp 4.250/Kg
                2. Jika jarak >10 km -->Rp 6.000/Kg
            4.MENGHITUNG TOTAL BIAYA = biaya per Kg x berat
            5.JIKA VOLUME > 1000.000 Cm --> biaya tamabahan Rp50.000 di tambahkan
            6.MENAMPILKAN VOLUME DAN TOTAL BIAYA.
